Fireflies — PyWeek Entry

Requirements
------------
- Python 3.7 or newer
- Pygame (pip install pygame)

How to Run
----------
Run:

    python run_game.py

This script checks your Python version and launches the game.

Controls
--------
- Left / Right arrows or A/D: Move
- Space: Jump (double jump available)

Goal
----
Collect all fireflies on each level, then reach the lamp post to advance.
Avoid incoming projectiles. Each level is procedurally generated.
